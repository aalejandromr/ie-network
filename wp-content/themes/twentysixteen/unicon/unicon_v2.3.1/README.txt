# Update to Version 1.3+ from an earlier version
------------------------------------------------------------------------

- In version 1.3+ you can have multiple widgets per column in the footer. 
It was needed to change the widget areas for this, so you might need
to reorder / add your widgets again in the appropriate widget areas under 
Appearance > Widgets.

# Update
------------------------------------------------------------------------

1) Just copy & paste the new "unicon" Folder from the Themes folder you've 
downloaded on Themeforest in wp-content/themes/ and completely overwrite 
the old "unicon" folder. Make sure to save your translation files before 
overriding with user translations.

2) Login to your WordPress Backend

3) Go to Appearance > Theme Options and hit "Save All Changes" just to be 
sure everything gets in the Database and looks good.

4) If there are any plugin updates, there should be a notice that guides 
you through the update installation of the plugins. Always update the plugins 
along with the theme to provide full compatibility. If you want to manually 
install the plugins you can find them in the theme folder under
/framework/plugins/

Notice: If you get a failure notice when you update the Visual Composer
Plugin, then deactivate your currently running VC plugin before updating.

------------------------------------------------------------------------
# Documentation
------------------------------------------------------------------------

To view the Theme Documentation, navigate to the documentation folder,
then to the "Theme Unicon" folder and open the "Documentation.html" file 
in your Browser. 

The "Icons-FontAwesome.html" and "Icons-SimpleLine.html" files are 
cheat-sheets for all the different icons that are included within
different elements of the Page Builder.

In the Visual Composer & Revolution Slider folder you can view the
Documentations of the included Premium Plugins.

# Theme Files
------------------------------------------------------------------------

- unicon.zip
This is the theme itself and should be uploaded to your 
WordPress Installation.

- unicon-child.zip
This is a sample child theme that you can use on your Installations.
It's just a blank canvas for you to start with your own Child Theme.
